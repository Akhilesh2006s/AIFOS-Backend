# AIFOS-Backend

NestJS API for AFIOS — enterprise operations platform (projects, supply chain, workforce, finance, mission control, explorer).

## Setup

```bash
npm install
cp .env.example .env
# Edit .env with MongoDB URI and JWT_SECRET
npm run start:dev
```

## Scripts

- `npm run build` — compile
- `npm run start:prod` — production server
- `npm test` — unit tests
- `npm run test:e2e` — API e2e tests
